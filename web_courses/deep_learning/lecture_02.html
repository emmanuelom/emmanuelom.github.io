Sección 01

¿Qué es un modelo preentrenado?

Un modelo preentrenado ya aprendió patrones generales a partir de una gran colección de datos. En esta clase, nosotros sólo haremos inferencia: pasar una imagen al modelo y leer su predicción.

🏋️
Entrenamiento
El modelo aprende durante muchas iteraciones usando imágenes etiquetadas. Este proceso requiere datos, tiempo y cómputo.
🧠
Modelo aprendido
Los pesos de la red almacenan patrones visuales: bordes, texturas, partes de objetos y combinaciones más complejas.
⚡
Inferencia
Usamos el modelo congelado para producir una predicción sobre una imagen nueva. No actualizamos los pesos.

Sección 02

ImageNet: el vocabulario visual del modelo

ResNet50 está entrenado para clasificar imágenes dentro de un conjunto cerrado de categorías. Por eso puede reconocer perros, autos o instrumentos, pero no necesariamente conceptos fuera de su vocabulario.

📚
1000 clases
La salida del modelo es un vector con 1000 valores: un score para cada categoría de ImageNet.
🎯
Clasificación
El modelo responde: “¿a cuál clase conocida se parece más esta imagen?”
⚠️
Dominio limitado
Si la imagen es médica, industrial o muy especializada, puede fallar porque no fue entrenado para ese dominio.
✓ inferencia✓ modelo congelado⚠ vocabulario fijo⚠ no segmenta
Sección 03

De imagen a tensor

Una red neuronal no recibe una imagen como archivo .jpg. Recibe un tensor numérico. En visión por computadora, el orden común en PyTorch es C × H × W.

Imagen RGB
H × W × 3
Tensor PyTorch
3 × 224 × 224
Batch
1 × 3 × 224 × 224
Sección 04

Pipeline de inferencia

El preprocesamiento es parte del modelo. Si cambiamos el tamaño, el orden de canales o la normalización, la predicción puede cambiar.

pipeline.py
# 1. Cargar imagen
image = Image.open("dog.jpg").convert("RGB")

# 2. Aplicar transformaciones del modelo
preprocess = weights.transforms()
x = preprocess(image)

# 3. Agregar dimensión batch
x = x.unsqueeze(0)  # [1, 3, 224, 224]

# 4. Inferencia
with torch.no_grad():
    logits = model(x)

# 5. Convertir scores a probabilidades
probs = logits.softmax(dim=1)
Sección 05

¿Por qué ResNet50?

ResNet introdujo conexiones residuales: atajos que permiten que la información fluya a través de muchas capas. Para esta clase basta entenderlo como una red profunda usada como clasificador general.

🧱
Bloques residuales
La red aprende una corrección sobre la señal original, en lugar de reconstruir todo desde cero.
📦
Feature extractor
Las primeras capas extraen patrones visuales. Las últimas producen la clasificación.
🎚️
Salida
Produce 1000 scores. Softmax los convierte en probabilidades comparables.

Sección 07

Código equivalente en PyTorch

La demo anterior corre en JavaScript para publicarse en GitHub Pages. En laboratorio o Colab, los alumnos pueden hacer lo mismo con PyTorch y Torchvision.

resnet50_inference.py
import torch
from PIL import Image
from torchvision.models import resnet50, ResNet50_Weights

weights = ResNet50_Weights.IMAGENET1K_V2
model = resnet50(weights=weights)
model.eval()

image = Image.open("mi_imagen.jpg").convert("RGB")
preprocess = weights.transforms()
x = preprocess(image).unsqueeze(0)

with torch.no_grad():
    logits = model(x)

probs = logits.softmax(dim=1)
top5_prob, top5_id = torch.topk(probs, 5)

categories = weights.meta["categories"]
for p, idx in zip(top5_prob[0], top5_id[0]):
    print(categories[idx], float(p))
